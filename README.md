Чтобы запустить локально - склонируйте репозиторий и следуйте инструкции:

В корне:

1. `pnpm install`
2. `pnpm dev`

Для запуска бэкенда:

1. `cd backend`
2. `pnpm install`
3. `pnpm start:prod`
